# SOLIDHINT


`SOLIDHINT` 工具纹理，当应用于一个刷子(brush)时，会告诉编译器不要切割其附近的其他面，从而提高性能。这在进行地形刷子工作时特别有用。

![未使用SOLIDHINT的原始地形工作，编译后](https://i.imgur.com/WlenfiV.jpg)
